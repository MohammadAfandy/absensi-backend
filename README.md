# ABSENSI BACKEND APP

> Example of a absensi backend APP using node, express, and mongoDB

## Quick Start

```bash
# Install dependencies
npm install

# Start Express Server: http://localhost:3000
npm start

# Build for production (Will build into server/public, ready for deployment)
cd client
npm run build
```
